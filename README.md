# Mongalet Dashboards

A landing page for Mongalet Dashboards that integrates a Droople awareness screen iframe.

## Structure

- `index.html` - Main landing page HTML
- `styles.css` - Styling for the landing page

## Features

- Responsive design that works on desktop and mobile devices
- Clean, modern UI with a gradient hero section
- Integrated Droople awareness screen dashboard
- Navigation menu
- About section

## Usage

Simply open `index.html` in a web browser to view the landing page.

## Customization

You can customize the styling by editing `styles.css` and modify the content by editing `index.html`.
